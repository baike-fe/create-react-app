# 动机
react-scripts并不支持多页面（多个html模版）应用程序，且有很多这样的需求， npm run eject 虽然支持自定义配置，但是如果有很多类似的项目，需要维护多份eject之后的配置项，所以fork出来一份，用于满足特定的需要。

## Feature

- 支持多页面
- 支持less
- 支持var.less
